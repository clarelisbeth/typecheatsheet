leading
line-height
This used to be the gap between lines of text - literally strips of lead to separate the lines. Now it means the measurement from the bottom to top of the glyph base thing.

line-height: 1.5rem;

Formal syntax: normal | | |

line-height: normal

line-height: 3.5 /* <number> values */

line-height: 3em /* <length> values */

line-height: 34% /* <percentage> values */

line-height: inherit

tracking
letter-spacing
The spaces between letters in a block - not between two specific characters.

letter-spacing: 1px;

Formal syntax: normal | <length>

letter-spacing: normal

letter-spacing: 0.3em

letter-spacing: 3px

letter-spacing: .3px

letter-spacing: inherit

dropcap
Large letter at the beginning of the copy intended to draw the reader in, and provide some typographic flair.
